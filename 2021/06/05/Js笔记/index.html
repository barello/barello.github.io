JS基础

JS编写位置

可以直接编写在标签内部（不常用）

写在<script>标签内

写在外部文件中，通过<script>标签引入

ps：当<script>标签用于引入外部文件时，就会忽略标签内部的代码。

JS变量

用var 关键词来声明一个变量

JS标识符

命名规则：

可以含有字母、数字、_、$,不能以数字开头

不能是关键字和保留字

命名规范：驼峰命名法

JS数据类型

基本数据类型的值直接在栈内存中存储

对象存储在堆内存

引用数据类型比较时比较地址

String 字符串

Number 数值

Number.MAX_VALUE 表示JS可以表示的最大数

当数值超过Number.MAX_VALUE时，会返回Infinity（正无穷 ）

Boolean 布尔值

Null 空

Undefined 未定义

Object 对象

数据类型转换

其他数据类型转换为String

1.调用被转换数据类型的toString（）方法，不影响原变量，而是将转换的结果返回。（null和undefined无toString（）方法）

2.调用String（）函数

其他数据类型转换为Number

1.调用Number（）函数（true为1，false为0）

2.parseInt（）把一个字符串转换为一个整数

parseFloat（）把一个字符串转换为一个浮点数

其他数据类型转换为Boolean

1.调用Boolean（）函数

非零数都为true

字符串中除了空串都为true

JS运算符

typeof 检查一个变量的类型

算术运算符不对原变量改值

JS对象

对象的分类

内建对象、宿主对象、自定义对象

宿主对象 BOM与DOM

创建对象

向对象中增加属性

对象.属性名 = 属性值

或者对象[“属性名”] = 属性值

ps:在[]中可以直接传递一个变量

作用域

全局作用域

在全局作用域中中有一个全局对象window（代表一个浏览器的窗口），创建的变量都会作为window对象的属性保存

变量的声明提前

使用var关键字声明的变量，会在所有代码执行前被声明（但不会被赋值）

函数的声明提前

用函数声明形式创建的函数function 函数（）{}会在所有代码执行前被创建

使用函数表达式创建的函数，不会被声明提前

创建函数

使用函数声明来创建对象

function 函数名（形参）{函数体}

使用函数表达式来创建一个函数

var 函数名 = function（形参）{函数体}

调用函数时，解析器不会检查实参的类型，也不会检查实参的数量
